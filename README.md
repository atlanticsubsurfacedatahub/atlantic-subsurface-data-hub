## Quick start
1) Open `index.html` in your browser.

   - Netlify (free)

## Customize fast
- Change the email in `index.html`:
  - `hello@atlanticsubsurfacedatahub.com`
- Change the logo text:
  - Find `ASDH` in the header.
- Change colors:
  - Edit CSS variables in `styles.css` under `:root`.
